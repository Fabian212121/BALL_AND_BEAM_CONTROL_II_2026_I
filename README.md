# BALL_AND_BEAM_CONTROL_II_2026_I
Este repositorio documenta el desarrollo de un sistema Ball and Beam como proyecto final de la asignatura Control 2, realizado por un equipo de cinco estudiantes de ingeniería. El objetivo principal es diseñar, modelar e implementar diferentes estrategias de control para estabilizar la posición de una esfera sobre una viga, un problema clásico en teoría de control que permite analizar sistemas dinámicos inestables.

El proyecto abarca desde el modelado matemático del sistema hasta la validación experimental. Se implementan y comparan tres enfoques de control: PID, compensadores de adelanto y atraso, y control óptimo LQR. Cada técnica es evaluada en términos de desempeño, estabilidad, tiempo de respuesta y error en estado estacionario, permitiendo identificar sus ventajas y limitaciones en aplicaciones reales.

Además, el repositorio incluye el diseño del sistema físico, la adquisición de datos mediante sensores, el acondicionamiento de señales y la implementación en tiempo real utilizando herramientas de programación y hardware de control. También se presentan simulaciones que respaldan el diseño antes de su implementación práctica.

Este trabajo busca no solo cumplir con los requisitos académicos, sino también desarrollar habilidades en análisis de sistemas, diseño de controladores y trabajo en equipo, integrando conocimientos teóricos con aplicaciones prácticas en ingeniería.
